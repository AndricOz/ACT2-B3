# ACT2-B3
Esta es mi segunda tarea de TALLER del 3er Bimestre, en Quinto Diversificado de Kinal
---------------------------------------------------------------------------------------------
Este proyecto simula el proceso de una venta sencilla:

A partir de una lista de productos con sus precios, calcula cuánto suma todo (el subtotal), 
cuánto hay que agregar de impuesto, y el total final que el cliente debe pagar. 

La idea es tener todo organizado y separado por partes, para que sea fácil de entender y de revisar.
